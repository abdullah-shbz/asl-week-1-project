# 🚀 Advanced CLI Calculator

This is a modular, interactive command-line calculator built in Python. It features robust error handling, an Object-Oriented math engine, and advanced functions like trigonometry and the quadratic formula. 

Built as the Week 1 Mini Project for the Advance Soft Logics AI/ML Internship.

## 🛠️ Setup Instructions

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone git@github.com:abdullah-shbz/asl-week-1-project.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd asl-week-1-project
   ```
3. **Ensure Python is installed:** This project requires Python 3.x. No external libraries are required (only the built-in `math` module is used).
4. **Run the application:**
   ```bash
   python main.py
   ```

## 💻 Usage

Upon running `main.py`, you will be greeted with an interactive terminal menu. Type the number corresponding to the operation you wish to perform (0-11) and press Enter.

**Features included:**
* Standard arithmetic (Add, Subtract, Multiply, Divide)
* Advanced Math (Power, Square Root)
* Trigonometry (Sine, Cosine, Tangent)
* Algebra (Quadratic Formula solver)
* Session History tracking

The application includes `try/except` validation to prevent crashes if letters or invalid characters are entered.
